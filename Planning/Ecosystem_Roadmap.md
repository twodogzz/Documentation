# Ecosystem Roadmap

This roadmap outlines the long-term direction of the SoftwareProjects ecosystem.

## Phase 1 — Stability and Automation (Current)
- Repo Manager stable and validated
- Push All Projects script operational
- Documentation structure established

## Phase 2 — Enhancements
- Add dry-run mode to Push All Projects
- Add colour-coded summaries
- Add commit message templates
- Add repo health checks

## Phase 3 — Integration
- Integrate Push All Projects into Repo Manager
- Add versioning automation
- Add new project detection

## Phase 4 — Evolution
- Port Repo Manager + Push All Projects to Python
- Package Python version as Windows EXE
- Add GUI interface
