# Master Project List Documentation

This folder contains the authoritative index of all software projects in the ecosystem.  
It provides a high-level overview, roadmap, task list, and change history.

## Files

- **MasterProjectList.rtf**  
  The master table of all projects, including ID, name, purpose, version, and last update.

- **ROADMAP.md**  
  Long-term direction and strategic goals for the entire ecosystem.

- **TASKS.md**  
  Actionable tasks, enhancements, and improvements.

- **CHANGELOG.md**  
  High-level record of major ecosystem changes.

## Purpose

This folder acts as the “control tower” for the entire SoftwareProjects environment.
